# CodeAlpha - Basic Chatbot

## Project Description

The Basic Chatbot is a simple rule-based chatbot developed using Python. It interacts with users by responding to predefined messages such as greetings, questions, and farewell messages. The chatbot uses conditional statements to generate appropriate responses.

## Objective

The objective of this project is to understand the fundamentals of chatbot development and improve Python programming skills using user input, loops, functions, and conditional statements.

## Tools Used

*  Programming Language : Python 
*  Editor : Visual Studio Code (VS Code)
* Module Used:
- Built-in Python Function
- String Handling Methods

## How to Run the Program

1. Install Python on your system.
2. Open the project folder in VS Code.
3. Open the terminal.
4. Run the following command:
     python chatbot.py
5. Start chatting with the bot by entering messages.
